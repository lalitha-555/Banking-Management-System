\# Banking Management System



\## Overview

A Java-based Banking Management System developed using Java, JDBC, and SQLite.



\## Features

\- Add Customer

\- View Customers

\- Search Customer

\- Update Customer

\- Delete Customer

\- Deposit Money

\- Withdraw Money

\- Check Balance

\- Transaction History



\## Technologies Used

\- Java

\- JDBC

\- SQLite
